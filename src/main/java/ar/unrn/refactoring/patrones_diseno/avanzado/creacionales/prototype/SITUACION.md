# Refactorizacion hacia Prototype

## Situacion inicial

El sistema crea escenarios de simulacion complejos copiando grafo, parametros y reglas manualmente en varias capas.

## Objetivo

Partir de prototipos ricos y clonar configuraciones base de manera controlada.

## Pasos para el vivo

1. Ubicar copias manuales extensas.
2. Dar al objeto la responsabilidad de clonarse.
3. Reutilizar prototipos preconfigurados como base.

## Dificultad

- Nivel: **Avanzado**
- Familia GoF: **Creacionales**
