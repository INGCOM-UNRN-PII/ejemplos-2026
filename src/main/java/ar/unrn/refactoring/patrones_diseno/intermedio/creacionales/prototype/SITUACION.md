# Refactorizacion hacia Prototype

## Situacion inicial

La aplicacion necesita crear objetos similares copiando manualmente muchos campos una y otra vez.

## Objetivo

Usar clonacion controlada para reutilizar objetos base ya configurados.

## Pasos para el vivo

1. Detectar copias manuales de objetos similares.
2. Mover la logica de clonacion al propio objeto.
3. Partir de prototipos preconfigurados.

## Dificultad

- Nivel: **Intermedio**
- Familia GoF: **Creacionales**
