# Refactoring

Esta carpeta queda organizada en tres ramas:

1. [`enunciados/`](./enunciados/README.md): ejercicios para resolver.
2. [`ejemplos/`](./ejemplos/README.md): ejemplos concretos de refactorizaciones ya armados.
3. [`patrones_diseno/`](./patrones_diseno/README.md): ejemplos orientados a refactorizaciones hacia patrones de diseno.

## Criterio de organizacion

- `enunciados` se mantiene separado porque su objetivo es practicar.
- `ejemplos` agrupa casos ya resueltos y ahora se ordena por dificultad.
- `patrones_diseno` queda preparado para crecer por dificultad sin mezclarlo con refactorizaciones generales.
