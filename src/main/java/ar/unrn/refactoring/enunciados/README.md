# Enunciados de refactorizacion

Este paquete agrupa ejercicios del tipo "dadas estas clases, aplica esta refactorizacion".

## Estructura

La organizacion es jerarquica:

1. dificultad (`basico`, `intermedio`, `avanzado`)
2. grupo conceptual
3. refactorizacion puntual
