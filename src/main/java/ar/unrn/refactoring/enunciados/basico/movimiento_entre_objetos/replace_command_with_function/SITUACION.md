# Replace Command with Function - nivel basico

## Enunciado

Dadas las clases de `Escenario.java`, aplica **Replace Command with Function**.

## Situacion inicial

- Clases involucradas: `CalculadorTotal`, `Pedido`.
- Olor o problema principal: Un objeto comando solo encapsula una operacion lineal y un resultado.
- Objetivo del cambio: Volver la operacion a una funcion mas directa.

## Consigna

1. Analiza las clases dadas.
2. Identifica donde aplicar la refactorizacion **Replace Command with Function**.
3. Reorganiza el codigo manteniendo el comportamiento observable.

## Nivel de dificultad

- Nivel: **Basico**
- Caracter del caso: Casos cortos, con una o dos clases y una sola decision principal de refactorizacion.
- Expectativa: Resuelve el ejercicio sin agregar capas nuevas si no son necesarias.

## Lo que se espera

- Identifica un unico punto principal de cambio.
- Aplica la refactorizacion sin reescribir todo el ejercicio.
- Manten el codigo resultante facil de leer.
