# Replace Type Code with Subclasses - nivel basico

## Enunciado

Dadas las clases de `Escenario.java`, aplica **Replace Type Code with Subclasses**.

## Situacion inicial

- Clases involucradas: `Empleado`.
- Olor o problema principal: Un codigo de tipo controla comportamiento y genera ramas condicionadas.
- Objetivo del cambio: Representar cada variante con un subtipo propio.

## Consigna

1. Analiza las clases dadas.
2. Identifica donde aplicar la refactorizacion **Replace Type Code with Subclasses**.
3. Reorganiza el codigo manteniendo el comportamiento observable.

## Nivel de dificultad

- Nivel: **Basico**
- Caracter del caso: Casos cortos, con una o dos clases y una sola decision principal de refactorizacion.
- Expectativa: Resuelve el ejercicio sin agregar capas nuevas si no son necesarias.

## Lo que se espera

- Identifica un unico punto principal de cambio.
- Aplica la refactorizacion sin reescribir todo el ejercicio.
- Manten el codigo resultante facil de leer.

## Pista docente

Busca el olor mas visible y resuelve primero el caso central.
