# Replace Function with Command - nivel basico

## Enunciado

Dadas las clases de `Escenario.java`, aplica **Replace Function with Command**.

## Situacion inicial

- Clases involucradas: `CalculadoraPrecio`, `Pedido`.
- Olor o problema principal: Una funcion concentra demasiados pasos y podria beneficiarse de un objeto con estado propio.
- Objetivo del cambio: Transformar la operacion en un comando explicito.

## Consigna

1. Analiza las clases dadas.
2. Identifica donde aplicar la refactorizacion **Replace Function with Command**.
3. Reorganiza el codigo manteniendo el comportamiento observable.

## Nivel de dificultad

- Nivel: **Basico**
- Caracter del caso: Casos cortos, con una o dos clases y una sola decision principal de refactorizacion.
- Expectativa: Resuelve el ejercicio sin agregar capas nuevas si no son necesarias.

## Lo que se espera

- Identifica un unico punto principal de cambio.
- Aplica la refactorizacion sin reescribir todo el ejercicio.
- Manten el codigo resultante facil de leer.

## Pista docente

Busca el olor mas visible y resuelve primero el caso central.
